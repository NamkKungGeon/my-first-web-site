HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearange of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive fors, may be embedded into the rendered page. It provides a means to create structured documents bydenoting structural semantics for text such as headings, pararaphs, lists, links, quotes and other items. HTML elements are delineated by tag, written using angle brackets.
